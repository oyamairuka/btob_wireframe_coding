## 概要

1. レスポンシブを一旦度外視して単純な要素配置ホーム画面実装
2. ホーム画面へのレスポンシブ設定
3. ヘッダーメニューなどの実装
4. 他画面への展開

この順序で進める予定（対応ブランチは以下）  

1. simple_arrangement_top_page
2. accessories_develop
3. accessories_develop（パンくずリストは他画面展開時に追加する）
4. 未定
